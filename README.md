# lenny

Lenny - Ideapad Dust Cleaner for Linux

There are four available modes:
* 0 -> Super Silent Mode
* 1 -> Standard Mode
* 2 -> Dust Cleaning
* 4 -> Efficient Thermal Dissipation Mode

## Installation

```
$ git clone https://github.com/maximyurevich/lenny
$ cd lenny
$ mv lenny.sh /bin/lenny
```
## Usage

```
$ sudo lenny 0
```
